How would you like the admin to be identified?

A

Hardcoded Email/UID

One specific Firebase Auth user (e.g., the owner's email) is the admin. Simple and secure for a single admin.

B

Role in Firestore

A users collection in Firestore with an admin role flag. Lets you grant admin access to multiple people later.

C

Secret URL / Toggle

No auth required — a hidden URL like /admin or a secret click pattern reveals the admin controls. Simplest but least secure.
